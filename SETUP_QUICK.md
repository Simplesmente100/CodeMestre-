# 🚀 Setup Rápido - Revit AI Assistant

## ⚡ Começar em 5 minutos

### 1. Obter API Key (GRATUITA)
1. Acesse: https://api.together.xyz/
2. Clique em "Sign Up" (criar conta)
3. Use seu email (não precisa cartão de crédito)
4. Após login, vá em "API Keys" 
5. Clique "Create new key"
6. **COPIE** a chave gerada

### 2. Instalar Plugin

#### Opção A: Build Automático (Recomendado)
```bash
# Clone o projeto
git clone [URL_DO_PROJETO]
cd revit-ai-assistant

# Execute o script de build
build.bat
```

#### Opção B: Manual
1. Baixe os arquivos compilados
2. Copie para: `%APPDATA%\Autodesk\Revit\Addins\2026\`
3. Reinicie o Revit

### 3. Usar no Revit
1. Abra Revit 2026
2. Procure aba **"AI Assistant"**
3. Clique **"AI Assistant"** 
4. Cole sua API key
5. **Pronto!** 🎉

"Como criar uma família de porta paramétrica?"

"Por que meu modelo está lento?"

## 🔧 Problemas?

- **Plugin não aparece**: Verifique pasta de instalação
- **Erro de API**: Teste a chave em https://api.together.xyz/
- **Erro de conexão**: Verifique firewall/proxy
